Help page

  • How To

    • Click on the toolbar addon and click on the MariMainPiano dropdown panel menu and it will open a piano window. There are two way to play this. One, is using the mouse to click one by one at the piano note interface. Two, is using the computer keyboard binding.

    Notice in MariMainPiano addon window there is two grey circular checkboxes with label of each one of it in the left corner of the screen. The key display labeled is for displaying what tone or note is the piano when we click one of them. The keyboard input labeled is to activate the keyboard binding with the piano note. To Activate this feature just click on one of the grey circle next to the label so it turns green.

    uncheck checked

    After keyboard input has been checked/activate just press on your computer keyboard to see how it works. The following input are describe in the picture.

  • Piano Chord

    • The chord is consist of three main musical note or more. And in order play them we should press more than one piano tone simultaneously. It is impossible to achieve this without activating the keyboard input so we should activate the keyboard input. Keep in mind that we play this in a computer keyboard environment so do not expect it would be as playable or produce great sound as the actual piano or in par.
